As part of my college work, I had an assignment to design and create a website for a fake cinema company called QuickTime Cinema. In this section I will show you the development process and the outcome.

To begin with, I created a first design for my website. I gave it a dark background and a purple theme. I kept fairly close to this design for the final product though there are some differences. In my design I also included a site map (which provides the layout for how the hyperlinks link to each other) the purpose and user requirements of the website and alternative designs to a few of the pages.

After I created my design I started work on creating the website. It consists of 8 different pages that all link to each other via the navbar: home, about, junior films, offers, news, members, login and register. I coded the website using Adobe Dreamweaver and made the the banner with Photoshop.

I think in the end I was fairly happy with how it turned out though it could have been improved, mainly in the design, but as a first website it wasn't so bad. A preview of the website can be found at weakeyes.dev/QuickTimeCinema.
